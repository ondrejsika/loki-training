# Loki Training: Log Management in Grafana

![Loki Reykjavik](_images/loki_reykjavik.png)

## Install Mac

```
brew install loki
brew install promtail
brew install logcli
```

## Install Linux (using slu)

```
slu ib loki
slu ib promtail
slu ib logcli
```
